How the journey began... and continues

  • In October 2019, I attended a coding workshop at Codaisseur where I got my first taste of programming by building an online game (the one you can find in my projects).
  • It's in January 2020 that I officially started my journey to learn code and enrolled in the Web Development path of Codecademy.
  • Via this online course, supplemented with many other resources, I've studied HTML, CSS and Javascript inside and out (and I still am).
  • Aside from those languages, I also learned how to use Git and the command line.
  • Other technologies I want to study further are Bootstrap, Sass and Android development.
  • In 2021, I'm brushing up the skills I've acquired so far, with FreeCodeCamp.
  • My goal this year is to become well-versed in Javascript and incorporate more interactivity in my websites!
